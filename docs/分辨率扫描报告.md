## 二、推理耗时

所有分辨率下推理时间稳定在 **71-77ms**，无明显差异。

说明 DSP 模型（det_500m / w600k_mbf）内部有固定输入尺寸，裁剪后会自动 resize 到模型原生输入，推理时间不随输入分辨率变化。

## 三、结论

| 指标 | 值 |
|------|-----|
| 最佳精度分辨率 | 900-1000（950 最优 score=0.837） |
| 推理耗时 | 71-77ms（各分辨率一致） |
| 裁剪带宽节省 | 全图 4MB → 裁剪约 0.7-2.5MB（取决于尺寸） |

综合精度和带宽，**推荐裁剪尺寸为 900-950**，在保持高精度的同时减少算法端输入数据量。
